# 🚀 Setup Paravibe Frontend

## 1️⃣ Clone Project

```bash
git clone https://github.com/LuanDHV/paravibe-fe.git
cd paravibe-fe
```

## 2️⃣ Install Dependencies

```bash
npm install
```

## 3️⃣ Setup Environment

Tạo file `.env.local` tại thư mục gốc:

```bash
NEXT_PUBLIC_API_BASE_URL=http://localhost:8080/api/v1
NEXT_PUBLIC_AI_API_BASE_URL=http://localhost:8000/api/v1
```

## 4️⃣ Start Frontend

```bash
npm run dev
```

✅ Frontend chạy tại: http://localhost:3000

---

## ⚡ Quick Commands

```bash
# Start dev (with hot reload)
npm run dev

# Build production
npm run build

# Start production
npm start

# Run linter
npm run lint
```

## 📋 Yêu Cầu
